```
`configs`:
    > Required

    The configs of the app.

    `name`:
        > Defaults to the index

        The name of the config.

    `apps`:
        > Defaults to empty list

        The apps

        `app_path`:
            > Required

            The path of the app

        `args`:
            > Defaults to empty list

            The Arguments/flags to start the app with

    `timeout`:
        > Defaults to 5
        > In seconds

        The timeout for number presses

    `keybindings`:
        See KEYMAP.md
```
